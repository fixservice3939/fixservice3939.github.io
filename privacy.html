Политика конфиденциальности

FixService (далее — «Сервис») уважает ваше право на конфиденциальность и обязуется защищать персональные данные, которые вы предоставляете при использовании сайта и формы заявки.

1. Какие данные собираются

При заполнении формы заявки на сайте вы добровольно предоставляете следующие данные:

  • • Имя
  • • Номер телефона
  • • Тип устройства
  • • Модель устройства (по желанию)
  • • Описание проблемы (по желанию)

2. Для чего используются данные

Собранные данные используются исключительно для:

  • • Связи с вами для уточнения деталей ремонта
  • • Консультации по услугам сервиса
  • • Записи на ремонт
  • • Формирования статистики (в обезличенном виде)

Данные не передаются третьим лицам, не продаются и не используются для спам-рассылок.

3. Хранение и защита данных

Ваши данные хранятся в защищённой базе, доступ к которой есть только у владельца сервиса FixService. Принимаются все необходимые меры для предотвращения несанкционированного доступа.

4. Срок хранения

Персональные данные хранятся до достижения цели их обработки (до выполнения заявки и закрытия диалога), но не более 1 года с момента последнего обращения. По запросу данные могут быть удалены досрочно.

5. Ваши права

Вы имеете право:

  • • Запросить информацию о своих данных
  • • Потребовать удалить свои данные из базы
  • • Отозвать согласие на обработку

Для этого свяжитесь с мастером: +7 (977) 811-16-66 или через Telegram: @AppleReanimator.

6. Изменения политики

Сервис оставляет за собой право вносить изменения в политику конфиденциальности. Актуальная версия всегда доступна на этой странице.
